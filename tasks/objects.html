Objects
Objects
Initalise an object with curly braces: { volume: 20, speaker: "Alexa" }
Useful for accessing properties and storing data associativity 
let obj = { volume: 20, speaker: "Alexa" };
console.log(obj);
Accessing object properties
Use the dot notation or array notation to access the properties 
let obj = { volume: 20, speaker: "Alexa" };
console.log(obj.volume); // outputs value associated to volume
console.log(obj["speaker"]); // outputs value associated to speaker
Array of objects
Access the object within the array with the index and then object property 
let objectArray = [
    {number: 12, address: "Round lane", post_code: "XS12 07E" },
    {number: 25, address: "Wood close", post_code: "TY27 8ER" },
    {number: 2, address: "Otley road", post_code: "NE02 9AL" }
];

console.log(objectArray[1].post_code);
