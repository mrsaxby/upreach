Functions
Code readability is key
The function can be written several ways but ultimately do the same thing
If you find yourself writing code repeatedly that is the same then create a function to reduce the amount of code
Acronym - DRY (Don't repeat yourself)
Functions can be used for recursion (i.e) calling the function repeatedly to act as a loop
public function addUp(value1, value2) {
    return value1+value2;
}

const addUp = (value1, value2) => {
    // good for multi line functions
    return value1+value2;
}

const addUp = (value1, value2) => return value1+value2;
addUp(10,5); // Calls the function and passes in the parameters
