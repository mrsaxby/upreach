Comments
Comments are used for describing the code and to make it more readable
// Some descriptive text that describes what the code is doing
// This function returns the result of two numbers multipled together
const multiply = (num1, num2) => num1*num2;
JavaScript comments can also be used to prevent execution
Comment out a single line: Double slashes: //
Comment out multiple lines: Slashes with an asterix: /* Are used for JS comments */
The line(s) will not be executed
//console.log("I wont run as I am commented out");
console.log("I will run as I am not commented out");

Variables
Variable names should be:
Unique
Must begin with a letter
Case sensitive
Usually camelCased
let
Usual declaration
let currentTime = new Date(Date.now());
const
Values that don't change (Number of months, Pi etc.)
const numberOfMonths = 12;

Data Types
string
Text wrapped around ('single' or "double quotes") will be stored as a string
let stringVal = "I am a string";
number
1234 - is a number
"1234" - is a string
let numberVal = 1234; typeof numberVal;
let numberVal = "1234"; typeof numberVal;
boolean
A boolean is either true or false
let booleanVal = false;
undefined
A variable that has been declared without a value will have the data type undefined
let undefinedVal;
object
Uses key:value to store associated values
let objectVal = {id: 1, name: 'Sonos Speaker', volume: 5 };
array
Stores multiple datatypes within the array
let arrayVal = ['string', 22, {id: 1, name: 'Sonos Speaker', volume: 5 }];
Console
Parameters can be passed into the console.log() function
This is useful for identifying the value of the variables
console.log("Output to the console");
console.table({shape: "cube", w: 5, h: 5, area: 25});
