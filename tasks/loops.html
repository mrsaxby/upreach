Loops
For loop
For loops do a loop x amount of times
Useful when the loop size is known 
// Increment i to loop forward
for(let i = 0; i < 10; i++) {
    console.log(i);
}

// Decrement i to loop backward
for(let i = 10; i > 0; i--) {
    console.log(i);
}
While loop
While loops do a loop until a condition is met
Useful when the loop size is unknown / searching for an element 
let x = 5;
while(x < 10) {
    console.log(x);
    x++;
}
Break
Cancel a loop when a condition is met 
let x = [1,2,3,4];
for(let i = 0; i < x.length; i++) {
    console.log(i);
    if(i === 2) {
        console.log("Broke out of loop on 3rd element");
        break;
    }
}
