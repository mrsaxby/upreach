Comparisons
Equal to AND Equal data type
Compares the value and the typeof value
Reduces bugs to use the three === comparisons
5 === 5 // true
5 === "5" // false
Equal to
Compares the value
5 == 5 // true
5 == "5" // true
Not equal to OR not equal data type
10 !== 5 // true
10 !== "5" // true
10 !== "10" // true
Not equal to
10 != 5 // true
10 != "5" // true
10 != "10" // false
Greater than / Less than
Greater than and Less than can be rearranged to achieve the same outcome
14 < 15 // 15 is greater 14
14 > 15 // 14 is less than 15
Greater than / Less than Equal to
Checks if the values are equal and less/greater than
15 < 15 // false
15 <= 15 // true
Conditionals
if
checks if the statement is true
Comment out multiple lines: Slashes with an asterix: /* Are used for JS comments */
The line(s) will not be executed 
if(5 < 10) {
    console.log('The statement is true');
    console.log(5 < 10); // outputs true
}
if else
checks if the statement is true
if false run the else statement 
if(15 < 10) {
    console.log('The if statement is true');
    console.log(5 < 10); // outputs true
}
else {
    console.log('The if statement is false');
    console.log(15 < 10); // outputs false
}
Switch statement
Similar to an if statement
Can improve code readability rather than having many if statements 
let variable = 2
switch (variable) {
    case 1:
        console.log("if variable == 1 call 1st function");
        break;
    case 2:
    console.log("if variable == 2 call 2nd function");
    break;
}

Ternary operator
Ternary operator
Similar to an if else statement
Useful shorthand
if(expression) ? true : false

if(variable > 10 || variable < 30) {
    console.log('The outcome is true as one of the statments == true');
