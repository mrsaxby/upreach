Arrays
A JS array can contain multiple data types.
The values in the array can be accessed by referring to the index
const exampleArray = [32,56,766,'LS2 87GH','12a road place', false, undefined];
Get the first element in the array
Indexes start at 0
exampleArray[0]
Get the second element in the array
exampleArray[1]
Get the third element in the array
exampleArray[2]
Get the fourth element in the array
exampleArray[3]
Count how many elements are in the array
exampleArray.length
Get the last element in the array
exampleArray[exampleArray.length - 1]
Array methods
join
Concatenates the array into a string
exampleArray.join(" and ");
filter
Returns a new array filled with values that meet the criteria
exampleArray.filter(arrayItem => typeof (arrayItem) === "number");
slice
Returns a new array with selected values from the array
exampleArray.splice(0,5);
forEach
Loops over each element within the array
exampleArray.forEach(arrayItem => { console.log(arrayItem)); });
map
Calls a function on element within the array
exampleArray.map(arrayItem => "<h1>"+arrayItem +"</h1>";
