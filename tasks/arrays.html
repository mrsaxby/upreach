    A JS array can contain multiple data types.
     - number (1,2,3,45,765)
     - string ("I'm a string", 'Ditto')
     - boolean (true, false)
     - null
     - undefined
     - object ({productName: "Sonos Speaker", volume: 12})

     
    let exampleArray = [32,56,766,'LS2 87GH','12a road place', false, undefined];

    Arrays start at index [0]
    exampleArray[0] = 32
    exampleArray[3] = 'LS2 87GH'
    exampleArray.length = 7 


    exampleArray[exampleArray.length - 1] = undefined
    
    exampleArray.push("newItem")
Access elements within the array with exampleArray[2]
COMMON ARRAY METHODS
Join
exampleArray.join(" - ");
Filter
exampleArray.filter(arrayItem => typeof (arrayItem) === "number");
Splice
exampleArray.splice(0,3);
Modifies array to contain only contain items from [0] to [3] // First 3 items
Slice
exampleArray.slice(0,3);
Same as Splice but does NOT modify array
forEach
exampleArray.forEach(arrayItem => { console.log(isNaN(arrayItem)); });
